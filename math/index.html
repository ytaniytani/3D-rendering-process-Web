2. なぜ学校の数学は怖かったのか

多くの人にとって、数学が苦手になった瞬間はよく似ています。「その記号が何のためにあるのか」を教わる前に、記号の操作だけを暗記させられたときです。sinとは何かを実感する前に加法定理を暗記し、行列が何をするものか知る前に掛け算の手順を覚えさせられた。意味の伴わない記号操作は、大人でも苦痛です。あなたが悪かったのではなく、順序が逆だったのです。

このトラックは順序を戻します。意味と実物が先、記号は後。あなたは本編で、回転も、遠近感も、光の当たり方も、すでにスライダーで触って知っています。つまり「実物」はもう手の中にある。ここでやるのは、その実物に正式な名前と表記を与えることだけです。

3. あなたとの5つの約束

  1. この扉は任意です。途中でやめて本編に戻っても、何も失いません。各ページに戻り口を用意しています。
  2. 一度に導入する新しい記号は1つだけ。新しい記号が出た段落では、他の新記号は出しません。
  3. すべての式は、本編で触ったデモと結びつけます。式が単独で登場することはありません。「あのスライダーがこの文字だ」を毎回やります。
  4. 導出や証明はしません。目標は式を「解ける」ことではなく「読める」こと。式の意味が言葉で言えれば合格です。
  5. 1画面に数式は1つまで。スクロールするたびに1歩ずつ。急かしません。

4. このトラックの道具: 「生きた数式」

7ページを通して、数式は必ず触れる状態で登場します。数式の項（式のパーツ）にマウスを乗せると、右の図の対応する部分が光ります。逆に図を操作すると、式の下の数値がリアルタイムに変わります。式と絵は、常に地続きです。

まず肩慣らしに、誰でも知っている式で道具の使い方だけ体験してください。数式の「縦」「横」「面積」にマウスを乗せてみる。それからスライダーを動かして、式の下の数値が追いかけてくるのを見る。この2つの操作が、この先ずっと使う基本動作です。
